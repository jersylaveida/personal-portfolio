# Personal Portfolio

A modern personal portfolio website for a software developer, built with Next.js, TypeScript, and Tailwind CSS.

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Notes

- The hero image is stored at `public/hero-workspace.png`.
- Add your real CV as `public/cv.pdf` so the download buttons work with your document.
- Main page content lives in `app/page.tsx`.
